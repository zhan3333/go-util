# 用范型来写一些工具函数与方法

go 版本最低需要 go1.18.1

goland 建议版本 2022.1, 该版本优化了很多泛型的提示。低于该版本 IDE 很有可能有语法报错。

官方示例: https://go.dev/doc/tutorial/generics

    - Empty()
    - Map()
    - Unique()
    - Merge()
    - Pad()
    - Append()
